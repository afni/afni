.. code-block:: none

    
    @NoPound [AFNI_FILES]
    Replaces all # characters in AFNI names with a -
    
    Example 1 :
     @NoPound elvis#001+orig.HEAD '*rockand#orroll*.HEAD'
    
    Example 2 :
     @NoPound 
        equivalent of @NoPound *#*.HEAD
